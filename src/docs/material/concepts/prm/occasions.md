# Occasions

Occasions are recurring events associated with [people](persons.md) in your life.

Each person can have multiple occasions. An occasion has a _name_,
a _date_ (like `12 Apr`), and a _kind_.

## Kinds

There are four kinds of occasions:

* **Birthday** - the person's birthday
* **Anniversary** - wedding anniversaries, work anniversaries, etc.
* **Holiday** - special holidays you celebrate with this person
* **Other** - any other recurring yearly event

## Task Generation

Inbox tasks are [generated](../tasks-generation.md) for each occasion,
similar to how [catch-ups](persons.md) work. The tasks appear on
the calendar and become actionable a few days before the
occasion date, giving you time to prepare.
