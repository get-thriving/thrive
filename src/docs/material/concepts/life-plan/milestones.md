# Milestones

A _Milestone_ is a named, dated event that marks an important moment in your
life. Milestones are part of the [Life Plan](overview.md).

## Purpose

Milestones capture the significant events that punctuate a life — the
moments you look back on as turning points or landmarks. Unlike
[chapters](chapters.md), which span a period of time, a milestone is a
point in time.

Examples of milestones:
- _"Graduated university"_ — June 2019
- _"Got married"_ — August 2022
- _"Moved to Berlin"_ — March 2024
- _"First child born"_ — November 2025
- _"Started company"_ — January 2026

## Properties

| Property | Description |
| -------- | ----------- |
| **Name** | A short label describing the milestone. |
| **Date** | The specific date on which the milestone occurs (past or future). |
| **Project** | The [project](projects.md) this milestone is associated with. |

Milestones also support a **note** for additional free-form context and
**tags** for organisation.

## Using Milestones in Chapters

Milestones can be used as anchor points for [chapter](chapters.md) start
and end dates. Instead of specifying a fixed calendar date for when a
chapter begins or ends, you can reference a milestone by name. This keeps
your chapters automatically consistent with the dates of your key life
events.

For example, a chapter _"Early parenthood"_ might start at the milestone
_"First child born"_ and end at _"Youngest child starts school"_.

## Accessing Milestones

Open the **Life Plan** section from the sidebar, then choose **Milestones**
from the menu.
