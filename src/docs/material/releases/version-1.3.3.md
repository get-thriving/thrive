# Version 1.3.3

Launched on 2026/05/27

This version introduces:

* History of changes to entities
* A tool to view "the weeks of life"
* A "todo" app as a replacement for "inbox tasks"
* A reworking of tasks and how they relate to other entities.
* More quotes
* Better search - better matching, inline search, etc.
* Ergonomics about delaying tasks
* Overdue signals on big plans too
* A life plan concept for planning life at a larger scale
* An MCP server - still very rough, but you can play with it and Claude/ChatGPT already
* An API - so you can script 
* Metrics direction - which direction is it good for the metric to go
* Exporting calendars as ical so other cal apps can use them
* Kanban views for big plans and time plans
* Ability to run on postgres (Even for self-hosted, but not documented)
* Separate crons that process various events (useful for visibility into self-hosted)
* Many bugfixes and small improvements
