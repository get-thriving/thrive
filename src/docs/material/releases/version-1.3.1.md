# Version 1.3.1

Launched on 2025/12/21

This version introduces:

* Add stats for a time plan, showing how much is planned, and what is
  realistic to do.
* Better support for self-hosting, with docs started as a service
  next to the others, so you can have your own version of docs.
* Loads of refactoring, to allow for better progress in the future.
* Docker images for amd64 and arm64 are generated now.
* Many bugfix, improvements, and polishing.
